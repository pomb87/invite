# invite
wedding invite template using nodejs, expressjs and ejs. Data is getting saved in sqlite3 db.

# setup
1. install express
npm install express
2. install ejs
npm install ejs
3. install sqlite3
npm install sqlite3
4. install express-generator
npm install express-generator -g
5. create invite folder
express invite

# start
node server.js

# on server
forever start server.js
